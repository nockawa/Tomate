Stores a string using UTF8 format with a max size of 256-bytes.

[PublicAPI]
public struct String256 : IComparable<String256>, IEquatable<String256>

Remarks

This struct stores 256 bytes as a fixed array of byte, the string is stores in the buffer encoded with UTF8. So the size of this struct is fixed and always of 256 bytes.
