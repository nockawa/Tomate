Stores a string using UTF8 format with a max size of 64-bytes.

[PublicAPI]
public struct String64 : IComparable<String64>, IEquatable<String64>

Remarks

This struct stores 64 bytes as a fixed array of byte, the string is stores in the buffer encoded with UTF8. So the size of this struct is fixed and always of 64 bytes.
