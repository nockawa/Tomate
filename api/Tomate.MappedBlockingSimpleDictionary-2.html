Remarks

Designed to be simple, accesses are O(n), a key of value default(TKey) is not permitted as it's used to detect free entries. Read-only access operations are made through a shared lock, read-write through an exclusive one. There can be multiple concurrent RO operations, but any RW operation will be exclusive and block any other RO/RW operations. You can enumerate the dictionary content, but only to evaluate it. Any call to ro/rw operations would lead to a deadlock.

Remarks

A shared lock will be created before enumeration and released at the end. The enumeration is not meant to change the content of the dictionary. Don't call any operation on the dictionary that leads to a shared/exclusive access, it would lead to a dead-lock as locks are not re-entrant on this type.
