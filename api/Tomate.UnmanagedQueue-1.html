Remarks

This class is heavily based from the Queue<T> type of the .net framework. Designed for single thread usage only. TryPeek(ref T) and Peek() return 

ref of {T}
, so don't use this reference after a Enqueue() or Dequeue() operation.

T

This method allocates the item in the queue and return a reference to it, it's up to the caller to set the value of the item. Don't keep the reference more than what is strictly necessary because it can be invalidated by operations that resize the content of the queue.
